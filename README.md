# Amigo Secreto

Challenge programa ONE.

## Descripción de funciones de la Carpeta app.js

### Función agregar amigo

```bash
function agregarAmigo() {
...
}
```
Esta función tiene como objetivoañadir un amigo, en la cual se valida el mensaje de un sorteo anterior, validar que el campo no este vacio; asi como agregar nombres al array, limpiar el campo de entrada y actualizar la lista en pantalla.

### Función actualizar lista

```bash
function actualizarLista() {
...
}
```
Esta función sirve para mostrar la lista de amigos en el HTML y convertirlos a una lista ordenada

### Función actualizar lista

```bash
function sortearAmigo() {
...
}
```
Esta función sirve para sortear un amigo, validar que haya al menos un amigo en la lista, generar un indice aleatorio, mostrar el recuadro y colocar el texto respectivo; por último, limpia el array y la lista para un nuevo sorteo
